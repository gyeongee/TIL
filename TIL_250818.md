- 큰 커널 : 큰 모양, 구조 파악 가능 ex) 얼굴 전체 윤곽
   but, 파라미터 수가 증가, 모델 과부하 가능성 존재

- 합성곱층은 이미지 특징을 선형으로 추출, 선형성을 가짐
- 비활성화 함수 사용 = 선형을 비선형으로 -> 복잡한 패턴 학습 가능
