### Django 설치 방법
```
# 1. 가상환경 생성
  python -m venv venv
# 2. 가상 환경 활성화
.\venv\Scripts\activate
# Django 설치
pip install django

# 현재 폴더에 프로젝트 생성
django-admin startprojext myproject . 

# 앱 생성
python manage.py startapp myapp

# 앱 생성 후에 프로젝트>settings.py에 앱 등록
INSTALLED_APPS = [
    ...
    'myapp',
]
```
